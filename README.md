# imersao-dados-com-python-Alura-2026
Análise de dados com python

Ao longo da imersão, foram trabalhados conceitos essenciais do ciclo de dados, incluindo:

 🔹 Python para análise de dados, com uso de bibliotecas como Pandas e NumPy para limpeza, transformação e exploração de dados
 
 🔹 Análise exploratória (EDA), identificação de padrões, outliers e geração de insights
 
 🔹 Visualização de dados, aplicando boas práticas de storytelling com gráficos claros e orientados à tomada de decisão
 
 🔹 Construção de um dashboard interativo, permitindo a publicação e apresentação dos resultados de forma acessível
 
 🔹 Uso de Google Colab como ambiente de desenvolvimento, facilitando reprodutibilidade e colaboração
 
 🔹 Introdução ao Streamlit, transformando análises em aplicações de dados simples e funcionais
